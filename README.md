🚀 Airpen — Draw in the Air Using Hand Gestures! ✨
Airpen is a real-time, AI-powered touchless drawing web tool that allows users to write, draw, and interact with a virtual canvas using only hand gestures ✋🎨. Instead of traditional inputs, this system completely removes the need for a mouse, trackpad, or physical touch screen.

🔹 Key Features
* 100% Touchless Interaction:** Write or sketch seamlessly directly through your webcam feed.
* Intelligent Framework:** Utilizes real-time hand landmark tracking to read your movements.
* Algorithmic Smoothing:** Built-in coordinate averaging filter to eliminate camera shake for clean strokes.
* Smart State Management:** Fully functional Undo, Redo, and Canvas Clear stacks with a PNG export feature.

🛠️ Technologies Used
* HTML5 & CSS3** (Frontend Structure & Layout)
* JavaScript** (Core Logic & State Management)
* MediaPipe Hands API** (Real-time Hand Landmark Tracking)

🎯 The Gesture Control Map
 👉 **Index Finger** ➔ Writing & Drawing Mode
 🖐️ **Open Palm** ➔ Drag & Move Elements Mode
 ✋ **Left Hand Joints** ➔ Dynamic Eraser Mode
 🤌 **Pinch Gesture** ➔ Precise Brush Size Control
